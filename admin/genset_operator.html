OPERATOR GENSET

PROSEDUR OPERATOR GENSET

1. Pengenalan


Sebelum menggunakan atau mengoperasikan peralatan atau instrument, baca petunjuk dengan seksama dan pelajari setiap fungsinya, hubungan dan istilah – istilah serta symbol – symbol.

2. Panel Genset


  • Panel Genset terdiri dari 3 cubicle ( A, B, C )
    A. No. 3 genset panel
    B. No. 4 genset panel
    C. Panel utama dan panel keluaran
    ( Panel No.3 serupa dengan panel No.4 )

  • Panel Genset No. 3 dan No. 4
    Baris pertama dari atas – Instrument untuk genset Out put
    A = Ampere meter
    V = Volt meter
    PF = Power Factor Meter
    Hhz = Frequency Meter
    W = Watt meter
    Baris kedua dari atas synchronizing instrument (Untuk penyelarasan secara manual )
    V = Volt meter from other generator
    Hhz = Frequency meter
    Baris ketiga dari atas :
    AS = Ampere meter selector switch ( Meter at first row )
    VS = Volt meter selector switch ( Meter at first row )
    30 = Alarm announcer mesin dan generator
    A = DC ampere meter dari battery charging dan discharging
    V = DC volt meter battery voltage ( DC supply, power untuk control )
    Baris keempat dari atas :
    P3 = Emergency Stop
    OS = Jenis pengoperasian – OFF, MANUAL, AUTO
    VR =Voltage rheostat, untuk mensetting voltase yang dikeluarkan oleh GEN
    CBS = Circuit breaker switch, untuk switching manual
    P1 = Engine start push button, untuk menghidupkan mesin secara manual
    ( Dalam module genset )
    P2 = Engine stop push button, untuk memberhentikan mesin secara manual
    ( dalam module genset )

  • Panel keluaran dan Panel Utama
    A = Ampere meter untuk 3 phase ( Total current / AM per supply dari generator )
    SIG = Ground Fault Relay
    OS = Jenis pengoperasian, Manual atau Auto
    LS = Lead selector, untuk memilih generator mana yang akan menyala terlebih dahulu dan akan menetap pada beban yang min (Master Running )

3. Jenis atau Type Pengoperasian


  • OS ( Operation Mode Switch ) digunakan untuk memilih pengoperasian genset secara MANUAL atau OTOMATIS. Saat posisi switch “ AUTOMATIC ‘, Saat posisi switch automatic load monitor system energize dan AMF ( Automatic Main Failure )
    Status stand by
    • OS1 ( Operation Mode Switch Genset No. 3 ). Pada Auto Mode genset no. 3 akan beroperasi atau berjalan secara otomatis dan pada “ Manual Mode “ genset no. 3 harus dioperasikan secara manual
    • OS2 ( Operation Mode Switch Genset No. 4). Pada Auto Mode No.4 akan beroperasi secara Automatic dan pada “ Manual Mode “ maka Genset No. 4 harus dioperasikan secara manual
    • LS ( Lead Selector Switch ) digunakan untuk memilih generator ACB ( Air Circuit Breaker ) mana yang akan ditutup terlebih dahulu dan tetap menutup pada saat beban minimum

4. Pengoperasian Genset Secara Manual


  • Operation Genset Secara Manual
    Putar semua OS ( Operation Mode Switch ) pada posisi manual ; perhatikan bahan bakar, air pendingin, minyak pelumas, battery. Jika semuanya baik, nyalakan kipas Ventilation

  • *Tekan Tombol “ Start “ pada generator yang akan dinyalakan
    Setelah generator mencapai kecepatan yang konstan ( Kecepatan Operation ), perhatikan ketinggian minyak pada sisi generator. Periksa voltase, Hz. Jika semua benar, generator siap untuk dibebankan
    • Control Waktu Pemberhentian ( STOP )
      Gunakan procedure yang sama dengan point ( * ) untuk generator yang lain
    • Penyesuaian Generator
      Pada tahap ini diperlukan perhatian penuh
        - Pertama check dan periksa synchronizer instrument atau meter, voltage, frequency pada generator harus sama.
        - Teruskan ke panel yang lain dan periksa system penyesuaian ( syncron ) dari lag ke lead. Usahakan membuat pointer tetap ditengah dengan merubah penyesuai frequency naik atau turun, dan pertahankan tetap ditengah kemudian matikan generator CBS ( Circuit Breaker Switch ). Sekarang ke dua generator beroperasi secara parallel, siap untuk dibebankan ( Siap dimasukkan pada panel emergency ) Operator harus mengisi dan mencatat kondisi dari genset tersebut.

5. Pemberhentian Genset Secara Manual ( OFF )


  • Pindahkan bebab dari genset ( Emergency ) ke PLN pada setiap gedung ( secara manual ) atau beban akan dipindahkan secara otomatis menggunakan
  • System ATS. Setelah generator dalam keadaan mati, matikan CBS kemudian tekan tombol stop. Generator akan berhenti setelah 3 menit ( waktu pendinginan ). Generator siap untuk difungsikan. ( Jika Synchrony manual )
