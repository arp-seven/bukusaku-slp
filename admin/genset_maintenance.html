MAINTENANCE GENSET

PROSEDUR MAINTENANCE GENSET

Peralatan yang dibutuhkan :

- Kunci ring 1 set
- Kunci pas 1 set
- Kunci sock 1 set
- Obeng ( +, - )
- Multitester
- Thermometer stick
- Pipet larutan H2 SO / air destileter
- BJ meter Accu
- Tang combination
- Grease Vaseline
- Kain lap / majun

1. Pemeriksaan system pendingin air


  • Periksa radiator dari kebocoran air pendingin ( Standart water 6,5 – 8,0 pH, kandungan cholorin kurang dari 1000 ppm )
  • Periksa ketinggian air ( Harus normal dan penuh )
  • Periksa lubang penutup yang ada di engine
  • Periksa drain valve di radiator
  • Periksa pipa – ppa flexible radiator dari kebocoran air

2. Pemeriksaan system pendingin oli dan pelumasan


  • Periksa level ketinggian oli di basin tap genset
  • Periksa kebocoran di system jaringan pipa pendingin oli
  • Standart oli API Service CD, SAE 40

3. Pemeriksaan system bahan bakar


  • Pemeriksaan instalasi bahan bakar dari daily tank hingga fuel pump dari kebocoran
  • Bahan bakar standart ASTM, D 975 grade no. 2D, karena dengan mutu bahan bakar yang baik akan optimal performanya, memperpanjang umur mesin dan emisi gas buang polusi akan terhindari
  • Periksa screen bahan bakar yang ada di fuel filter
  • Periksa dan kendorkan plug air vent di atas tutup filter bahan bakar sampai udara terbuang dan keluar cairan bahan bakar
  • Buang larutan bahan bakar yang tercampur dengan air dengan membuka valve fuel tank di bawah daily tank / storage tank

4. Pemeriksaan battery


  • Periksa level cairan elektrolit battery dan tambahkan cairan elektrolit sampai level normal
  • Periksa system charging battery dari tingginya temperature cairan elektrolit H2 SO4 battery ( Full charging 50° C ) dan check berat jenis larutan elektrolit tersebut 1230 s/d 1250
  • Kencangkan vent plug
  • Bersihkan terminal battery dan berikan grease / Vaseline

5. Pemeriksaan dengan test running genset


  • Start engine melalui panel genset dengan menekan tombol running test ( position selector switch di posisi automatic )
  • Pastikan indicator Led charging di genset panel kecil OFF setelah genset running
  • Periksa frequency meter, harus menunjukkan ± 50 Hz
  • Periksa volt meter, harus menunjukkan ± 380 volt
  • Periksa indicator tekanan oli ( 2,5 ~ 6 kgf/cm )
  • Periksa indicator ampere ( sesuai load )
  • Operasikan genset 3 s/d 5 menit untuk trial test
  • Test juga emergency tombol stop

- Engine oil low pressure
- Engine coolant temperature low
- Over speed
- Emergency stop
- Penggantian oli
- Pembersihan air cleaner / filter udara
- Van belt
- Penggantian filter oli
- Penggantian filter bahan bakar
- Check altenator
- Check starter dan check solenoid start
